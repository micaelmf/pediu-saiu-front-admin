Textarea

Use textarea element as a multiline text input.

Use the textarea element instead of the input text element to receive longer data from users. The usage is similar to the text input element. The difference is only the element between an input and a textarea. So, we just provide basic examples on this page because of similarities.

Auto Grow #

In order to use this feature, you have to import the javascript manually because it is not a part of the core components.

Just wrap the textarea element with a div that has a data-textarea-grow data attribute.
